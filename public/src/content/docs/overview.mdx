AgentZero is a **lightweight, security-first agent runtime and CLI** built entirely in Rust. It provides a single-binary runtime with trait-driven extensibility, encrypted storage, and fail-closed security defaults.

:::note
AgentZero is under active development. The architecture is stable but individual features may change between releases.
:::

## Why AgentZero

- **Learn by building.** Clean crate boundaries make the entire system inspectable and understandable.
- **Security is not optional.** Scoped tools, allowlists, audit logging, and encrypted storage are baseline — not plugins.
- **No lock-in.** Swap providers, memory backends, tools, and channels through trait interfaces.
- **Deploy anywhere.** Single native binary. No container, no runtime, no external dependencies.

## Quick Start

```bash
# Build from source
cargo build -p agentzero --release

# Run interactive onboarding
cargo run -p agentzero -- onboard --interactive

# Set your API key
export OPENAI_API_KEY="sk-..."

# Send a message
cargo run -p agentzero -- agent -m "hello"

# Check system health
cargo run -p agentzero -- doctor models
cargo run -p agentzero -- status
```
