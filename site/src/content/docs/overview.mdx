AgentZero is **fast, small, and fully autonomous AI personal assistant infrastructure**, built entirely in Rust. ANY OS, ANY PLATFORM — deploy anywhere, swap anything.

:::note
AgentZero is under active development. The architecture is stable but individual features may change between releases.
:::

## Why AgentZero

- **Deploy anywhere.** A single native binary runs on Linux, macOS, and Windows — 8 platform targets, CI-tested on all three OS families. Docker multi-arch for containers. 4.5 MB embedded binary for resource-constrained devices.
- **Swap anything.** LLM providers, storage backends, tools, and channels are all swappable through trait interfaces. 37+ providers, 25+ channels, 58+ tools. Your agent logic stays the same.
- **Fully autonomous.** Three autonomy levels, autopilot with cap gates and missions, cron scheduling, multi-agent delegation and swarm coordination. Agents propose, approve, and execute work independently.
- **ANY PLATFORM.** Client SDKs for Python, TypeScript, Swift, and Kotlin talk to the gateway over HTTP/WebSocket. OpenAI-compatible API works with any existing client library. Interactive API docs at `/docs` on every deployment.
- **Secure by default.** Encrypted storage, allowlists, policy gates, audit logging, secret redaction, and emergency stop. Fail-closed from day one.

## Quick Start

```bash
# Build from source
cargo build -p agentzero --release

# Run interactive onboarding
cargo run -p agentzero -- onboard --interactive

# Set your API key
export OPENAI_API_KEY="sk-..."

# Send a message
cargo run -p agentzero -- agent -m "hello"

# Check system health
cargo run -p agentzero -- doctor models
cargo run -p agentzero -- status
```
