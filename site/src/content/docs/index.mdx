## Why AgentZero?

<CardGrid stagger>
  <Card title="Safety First" icon="approve-check">
    Every tool call goes through policy evaluation. Deny by default. Fail closed on unknown.
  </Card>
  <Card title="Local First" icon="laptop">
    All inference runs locally (Ollama, llama.cpp, vLLM, LM Studio). Remote calls require explicit policy and redaction.
  </Card>
  <Card title="MCP Server" icon="puzzle">
    Expose tools to Claude Code, Cursor, or Zed. Any MCP client gets policy-controlled access.
  </Card>
  <Card title="Encrypted Vault" icon="setting">
    Secrets stored with AES-256-GCM. Models see handles, never raw values.
  </Card>
</CardGrid>

## Quick Start

```bash
# Install
cargo install --path crates/agentzero-cli

# Initialize a project
agentzero init --private

# Chat with a local model
agentzero chat --local

# Start MCP server for your editor
agentzero mcp

# Scan for secrets and PII
agentzero run repo-security-audit
```
